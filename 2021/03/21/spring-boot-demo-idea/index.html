使用IDEA创建SpringBoot项目

2021-03-21

Step 1

首先，新建一个项目，项目类型选择Spring Initializr。如果 Project SDK 不存在的话，可以点击旁边的按钮选择自己电脑上安装的JDK的安装路径；或者点击下拉列表，选择自己想要的JDK版本。其它默认就好，接着点击下一步。

Step 2

接着就要设置这个项目的基本信息了。如下表所示：

英文 中文 说明
Group 开发此项目的组织 一般为域名的倒写
Arifact 项目的标识
Type 项目类型 Maven 或者 Gradle
Language 项目使用的编程语言 有 Java、Kotlin 等
Packaging 项目的打包方式 Jar 包 或者 war 包
Java Version JDK 的版本
Version 此项目的版本
Name 项目名
Description 项目描述
Package 包名

Step 3

这一步要添加项目的依赖了。如下图所示，可以直接选择自己需要的依赖项，或者在搜索框中进行搜索。因为这是我们初学 Spring Boot , 所以选择一个 Spring Web 依赖项就够用了。在窗口右上方可以选择 Spring Boot 的版本。

Step 7

接着，我们可以打开项目中的 Application 类，类似下图这样，添加一个返回 Hello World! 的 hello 方法, 并加上相应的注解。然后，就可以点击运行按钮，开始运行这个项目了。下方的控制台中会输出一些相应的信息。

Step 8

在上一步的控制台输出信息中，出现Started DemoApplication in 1.524 seconds这样的字样，就说明项目已经启动成功了。同时，还能看到项目内置的 Tomcat 服务器的默认端口号是 8080 。此时，我们可以打开一个浏览器，来验证项目是否能够正常工作。如下图所示，我们可以在浏览器的地址栏中输入 localhost:8080/hello , 页面正常显示出了 Hello World! ，大功告成！
