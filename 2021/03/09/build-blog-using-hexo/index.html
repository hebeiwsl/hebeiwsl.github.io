2021-03-09

前期准备

安装Git

因为 Hexo 的源代码还有主题都放在开源网站 Github 上，所以在搭建博客的过程中，我们会经常和它打交道。因此，安装一下 Git 会让我们轻松一些。我们可以在它的官网下载, 然后安装。

安装Node

Hexo 博客本身就是基于 Node.js 开发，所以如果没有安装 Node.js ，Hexo 根本无法运行。你可以去官网下载。安装完成后，你可以打开命令行窗口，输入以下命令来检查 Node.js 是否安装成功
